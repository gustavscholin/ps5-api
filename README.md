# PS5 Availability API
An API that checks various swedish online retailers for stock status of PS5.

## Online retailers included

- [Media Markt](https://www.mediamarkt.se/)
- [Elgiganten](https://www.elgiganten.se/)
- [Inet](https://www.inet.se/)
- [Webhallen](https://www.webhallen.com/)
- [NetOnNet](https://www.netonnet.se/)
